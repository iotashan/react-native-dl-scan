# Code of Conduct

## Our standard

This project is a place where everyone — contributors, users, maintainers — can
participate productively. We expect everyone to be respectful, patient, and
constructive in all project spaces, including issues, pull requests,
discussions, and any other communication channels.

## Examples of behavior we encourage

- Constructive feedback, focused on the work
- Acknowledging that other contributors have different experience levels and
  backgrounds
- Patience when explaining concepts to newcomers
- Crediting others' contributions

## Examples of behavior we do not accept

- Personal attacks, insults, or derogatory comments
- Targeted harassment, public or private
- Posting others' personal information without permission
- Sustained disruption of discussions or reviews
- Other conduct that a reasonable person would consider inappropriate in a
  professional setting

## Scope

This applies in all project spaces (GitHub issues, pull requests, discussions,
the example app, any chat or forum tied to the project) and in any public
communication that represents the project.

## Reporting

If you experience or witness unacceptable behavior, please email
**shan@iotashan.com**. Reports will be reviewed and responded to within seven
days. Confidentiality will be respected to the extent possible.

The project maintainer will determine appropriate responses, which may include
a private warning, a public note, a temporary restriction from project spaces,
or a permanent ban for serious or repeated violations.

## Acknowledgement

This Code of Conduct is intentionally short. It is inspired by the
[Contributor Covenant](https://www.contributor-covenant.org/) but written in
plainer language for clarity.
